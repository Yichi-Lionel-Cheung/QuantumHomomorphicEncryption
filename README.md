# Homomorphic encryption

see Notion-page `Tasks/Dual HE & HE`

install

```bash
## download the repo
git clone git@github.com:ClonelessLab/homomorphic-encryption.git

## for general usage
pip install .

## for developer
pip install -e ".[dev]"
```

Run unittest
s
```bash
pytest
```

Build docs

```bash
mkdocs serve
```
